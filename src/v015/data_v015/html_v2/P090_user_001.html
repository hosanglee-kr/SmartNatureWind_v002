Smart Wind

도움말 및 시스템 정보

💡 프로젝트 개요

Smart Nature Wind 시뮬레이터는 **자연풍의 복잡한 움직임(난류, 돌풍, 열기포)**을 정밀하게 재현하여 사용자에게 실제와 같은 환경 테스트를 제공하기 위해 개발되었습니다.

💻 기술 스택
  • **하드웨어:** ESP32 (Wi-Fi, Web Server)
  • **백엔드/펌웨어:** PlatformIO, Arduino Core (C++/INO)
  • **프론트엔드/UI:** HTML5, CSS3 (공통 스타일), JavaScript (Vanilla JS)
  • **차트:** Chart.js, chartjs-plugin-zoom
  • **통신:** RESTful API (설정), WebSocket (실시간 데이터)
📚 사용 가이드 요약
1. 초기 설정

▶️ **시스템 설정** 페이지에서 **API Key**를 설정하고, **네트워크 연결**을 확인하세요.

2. 풍속 모니터링

📈 **차트 모니터링** 페이지에서 실시간 **풍속** 및 **PWM 듀티**의 변화를 확인할 수 있습니다.

3. 프로파일 설정

🗂️ **프로파일 관리**에서 원하는 풍속의 강도, 가변성, 난류 특성을 저장하고 즉시 **적용**할 수 있습니다.

4. 스케줄 자동화

⏰ **스케줄 관리**에서 특정 요일/시간에 프로파일이 자동으로 적용되도록 예약할 수 있습니다.

5. 상세 조정

⚙️ **시뮬 설정**에서 시뮬레이션 간격, 돌풍/열기포 간격 등 디테일한 수치를 조정할 수 있습니다.
